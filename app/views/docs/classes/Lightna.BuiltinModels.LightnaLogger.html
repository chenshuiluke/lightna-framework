\Lightna\BuiltinModelsLightnaLogger

Model abstracts a lot of database CRUD functionality so the user won't need to write any SQL. Any user-created models should extend this class.

create()

create(array  $keyValArray)

Initializes items in the field list with values from $keyValArray. By the time this is called, some of the creator methods (i.e. string(), boolean(), int()) would have been called and added empty fields to the field array. So this function just takes an array of new field names and values and initializes those existing fields.

Parameters

array $keyValArray

An associative array containing the names of fields to initialize and the values to initialize them with.
